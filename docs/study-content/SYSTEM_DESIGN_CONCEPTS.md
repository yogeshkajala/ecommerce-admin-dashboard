# System Design Modules & Categories

This table maps the core system design concepts into distinct categories, assigned with relevant icons. These categories will serve as the foundation for separate information modules within the dashboard.

| Icon | Category | Core Concepts / Items |
| :---: | :--- | :--- |
| 🏗️ | **Core Fundamentals** | Scalability, Availability, Reliability, Latency, Throughput, Bandwidth, Client-Server |
| 🗄️ | **Data & Databases** | Databases, SQL vs NoSQL, Indexing, Sharding, Denormalization, ACID, BASE |
| 🏛️ | **System Architecture** | Monolith, Microservices, Event-Driven Architecture, BFF (Backend for Frontend), Service Mesh, Sidecar |
| 🔌 | **APIs & Communication** | API Design, REST, GraphQL, gRPC, WebSockets, WebRTC |
| 🌐 | **Distributed Systems** | CAP Theorem, Consistency Models, Replication, Partitioning, Consensus, Leader Election |
| 🚀 | **Performance & Scaling** | Caching, Cache Invalidation, CDN, Load Balancing, Autoscaling, Load Shedding |
| 🛡️ | **Reliability Patterns** | Fault Tolerance, High Availability, Retry Logic, Timeout, Circuit Breaker, Bulkhead |
| 🔒 | **Security** | Authentication, Authorization, RBAC, SSO, Encryption, Secrets Management |
| 📨 | **Messaging & Async Systems** | Message Queues, Pub/Sub, Sync vs Async, Idempotency, Backpressure |
| 👁️ | **Observability** | Monitoring, Logging, Metrics, Tracing, Alerting, Correlation IDs |
| 🔍 | **Data Processing & Search** | Batch Processing, Stream Processing, MapReduce, ETL, Full-Text Search |
| 🌳 | **Advanced Data Structures** | Bloom Filters, HyperLogLog, Merkle Trees, LSM Trees, B-Trees |
| 📊 | **Modern Data Platforms** | Data Pipelines, Data Lakes, Data Warehouses, Vector Databases |
| 🚢 | **Deployment Strategies** | Blue-Green Deployment, Canary Releases, Feature Flags |
| ⚙️ | **Additional Gateway/Cache**| Rate Limiting, API Gateway, Service Discovery, Cache Warming, Cache Stampede, CDN Caching, Serialization, Deserialization |

*Note: Icons are represented via Unicode emojis to ensure native rendering across the dashboard UI and markdown viewers.*
